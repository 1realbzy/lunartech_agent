# Transcripts Directory

This directory will store the detailed transcripts of all interviews conducted with the LunarTech Interview Agent.

Each transcript file is named with the format: `transcript_YYYYMMDD_HHMMSS.txt`

The transcript files include:
- Timestamp of the interview
- All questions asked by the agent
- All answers provided by the candidate, with timestamps
- Any clarification requests and follow-up responses
- Any FAQ questions asked by the candidate and the agent's responses

These files are plain text and can be opened with any text editor.
